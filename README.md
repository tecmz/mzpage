# 魔众小程序单页系统

魔众小程序可视化单页系统可以快速帮助可以搭建一个小程序单页系统，支持微信、抖音等平台

## 运行环境


```
操作系统
Linux/Unix 或&nbsp;&nbsp;Windows
软件环境
Laravel 5.1的运行环境
Apache/Nginx , PHP 5.5.9+ / PHP 7.0 , MySQL 5.0+
```
